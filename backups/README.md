# Backups

This directory stores generated local backups for the project.

What should go here:

- backend SQLite snapshots
- local `.env` copies
- local Kubernetes secret env files
- Docker volume archives for backend and monitoring data

What should not go here:

- source code backups for tracked files
- `node_modules`
- build output that can be recreated

This directory is gitignored except for this README.
